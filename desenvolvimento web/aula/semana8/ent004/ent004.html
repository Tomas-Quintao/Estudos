Comandos Essencias de Movimentação

  • Mover para cima:tecla / Console:
  • Mover para baixo: Tecla / console
  • Mover para Esquerda: Tecla / Controle
  • Mover para Direita : Tecla / console
  • Pular / Confirmar: ou enter / console: (x)
  • Ação Secundaria / cancelar: Tecla / console: (Bola/circulo)
  • Menu / Opções: Tecla esc / Console: (triangulo)
  • Inventario: Tecla I / console: □ (quadrado)

Sequência de Ataque Especial (Combo "Fúria Pixelada")

  1. pressione □ □ (duas vezes rapidamente)
  2. Em seguida, pressione ○ com precisão
  3. Finalize segurando R2 e apretando X
